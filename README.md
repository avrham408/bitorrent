# AVTorrent

Python implimentation for bitorrent client

## Getting Started

Have Python3.6.8+

```
pip install requirements.txt
```

To download torrent-

```
python avtorrent.py [torrentfile.torrent]
```


## Running the tests

```
pytest tests/
```
